Инструменты

Мы будем использовать следующее ПО:

  1. Компилятор C++: g++ или clang++.
  2. Библиотека SDL2.
  3. Опционально библиотека SDL2_mixer для звука.
  4. Библиотека WHY_CPP
  5. Утилита сборки проектов CMake и Make.
  6. Среда разработки CLion.
